# lws-media-player

# Supported Devices
- All Android brand

# Supported Feature
- play sound
- stop sound

# Requirement to build
- minimal version API 16 (Jelly Bean 4.1)

# Instruction to build

Pre ->
 - clone the git first
 - change branch to develop

On ->
 - open android studio
 - rebuild the project(menu build > rebuild)
 - hit the play button on android studio or (menu run > run app)
